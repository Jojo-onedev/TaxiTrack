# TaxiTrack

TaxiTrack est une plateforme complète de transport VTC conçue pour optimiser la mise en relation entre chauffeurs et clients, tout en offrant une gestion centralisée pour les administrateurs.

## Structure du Projet

Le projet est divisé en trois modules principaux :

- **/backend** : API robuste gérant la logique métier, l'authentification et l'interaction avec la base de données PostgreSQL.
- **/frontend-admin** : Tableau de bord web intuitif pour l'administration de la flotte, le suivi des courses et la gestion financière.
- **/mobile-app** : Application mobile hybride (Flutter) offrant une expérience fluide pour les clients et les chauffeurs.

## Objectifs Globaux

- Fournir une solution de transport fiable et sécurisée.
- Faciliter la gestion opérationnelle et financière de la flotte.
- Offrir une interface utilisateur moderne et réactive sur tous les supports.
