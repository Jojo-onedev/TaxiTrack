# TaxiTrack - Application Mobile

## Rôle du Module

Le dossier `mobile-app` contient l'application unique destinée à la fois aux clients et aux chauffeurs.

## Spécifications Techniques

- **Technologie** : Flutter (Single codebase).
- **Fonctionnalités clés** :
  - Interface dédiée pour les clients (réservation, suivi GPS).
  - Interface dédiée pour les chauffeurs (acceptation de courses, navigation).
  - Notifications push en temps réel.
  - Messagerie intégrée et système d'évaluation.
