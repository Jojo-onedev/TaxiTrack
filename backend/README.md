# TaxiTrack - Backend

## Rôle du Module

Le dossier `backend` contient l'API centrale de la plateforme TaxiTrack.

## Spécifications Techniques

- **Technologie** : Node.js avec le framework Express.js.
- **Base de données** : PostgreSQL.
- **Fonctionnalités clés** :
  - Gestion des utilisateurs (Clients, Chauffeurs, Admins).
  - Traitement des demandes de trajets en temps réel.
  - Calcul des tarifs et gestion des paiements.
  - API REST pour les applications web et mobiles.
